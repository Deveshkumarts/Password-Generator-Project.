# 🔐 Password Generator Project

A tool that generates strong, random passwords based on user-selected criteria.

## 🚀 Features
- User-defined password length
- Option to include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Simple CLI interface
